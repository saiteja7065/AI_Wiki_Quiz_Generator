# AI Quiz Generator - Frontend

React frontend for the AI Quiz Generator.

## Setup

```bash
# Install dependencies
yarn install

# Start development server
yarn dev

# Build for production
yarn build
```

## Tech Stack

- React 19
- Vite
- Tailwind CSS
- Yarn

## Components

- `App.jsx` - Main app with tab navigation
- `GenerateQuizTab.jsx` - Quiz generation interface
- `HistoryTab.jsx` - Quiz history viewer
- `InteractiveQuiz.jsx` - Interactive quiz component
- `QuizDisplay.jsx` - Study mode quiz display
- `Modal.jsx` - Modal component

## Configuration

Backend API URL: `http://localhost:5000` (configurable via `.env`)

## Development

The app runs on http://localhost:5173/ with hot module replacement enabled.
